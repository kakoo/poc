딥링크 : 고객을 '앱 안의 정확한 그 화면'까지 데려다주는 링크

웹 링크가 웹페이지를 여는 것처럼, ViaLink 링크는 앱의 특정 화면을 엽니다. 앱이 없으면? 스토어로 보내 설치시키고, 설치가 끝나면 원래 보여주려던 그 화면을 자동으로 엽니다.

01

광고를 클릭하면 바로 그 화면으로

광고 속 상품·이벤트 화면으로 즉시 연결됩니다. 앱 첫 화면에서 헤매다 이탈하는 고객이 사라집니다.

02

앱 미설치 고객도 바로 원하는 화면으로

스토어 → 설치 → 그 화면 자동 오픈. 디퍼드 딥링크가 설치 후에도 목적지를 기억합니다.

03

추천인 코드 자동 적용

코드를 직접 입력하는 이탈 포인트가 사라집니다. 링크만으로 추천인이 자동 인식되고 보상까지 이어집니다.

04

클릭 → 설치 → 결제까지 성과 측정

전 구간을 하나의 파이프라인으로 연결합니다. 캠페인 ROI를 추정이 아니라 데이터로 증명하세요.

05

QR 코드를 앱에 연결

오프라인 매장·포스터·패키지의 QR 코드도 딥링크가 됩니다. 스캔 한 번에 앱 안의 정확한 화면으로.

06

소셜 미디어 사용자를 앱 사용자로 전환

인스타그램·유튜브 프로필과 게시물에 건 링크가 앱 설치와 전환으로 이어집니다. 채널별 성과도 비교할 수 있습니다.

인플루언서·제휴 마케터와 연결하는 홍보파트너 기능

파트너에게 전용 링크를 발급하고, 파트너별 설치·결제 기여를 정산 가능한 데이터로 확인합니다.

인앱 브라우저에서도 열리나요?

iOS Universal Links, Android App Links를 기본 지원하고, 카카오톡·인스타그램·LINE 등 인앱 브라우저를 자동 감지해 각 환경에서 동작하는 최적 경로(스킴 폴백 포함)로 라우팅합니다. apple-app-site-association, assetlinks.json 호스팅은 ViaLink가 대신 처리합니다.

앱이 설치 안 된 유저는 어떻게 매칭하나요?

디퍼드 딥링크: 클릭 시점의 디바이스 시그니처를 서버에 저장하고, 설치 후 최초 실행 시 핑거프린트 매칭으로 원래 목적지 화면을 복원합니다. 매칭 유효 기간 72시간. IDFA/GAID 없이 동작하므로 ATT 환경에서도 유효합니다.

클릭 → 결제까지 어떻게 이어서 보나요?

track() 한 줄로 결제 시도·결제 성공 이벤트를 심으면, 클릭 → 설치 → 결제 시도 → 결제 성공이 라스트 터치 기준 단일 파이프라인으로 귀속됩니다. 대시보드에서 채널·캠페인·링크별 전환 퍼널을 바로 비교할 수 있습니다.

ViaLinkSDK.shared.onDeepLink = { link in router.navigate(to: link.path, params: link.params) } ViaLinkSDK.shared.track("purchase_success", data: ["amount": 12900])

크레딧을 모두 소진하면 딥링크 기능이 바로 중단되나요?

아니요, 즉시 중단되지는 않습니다. 크레딧이 소진된 후에도 고객님의 서비스가 원활하게 운영될 수 있도록 충분한 유예 기간을 드립니다. 해당 기간 동안은 알림을 통해 크레딧 충전이 필요함을 안내해 드리며, 일정 시간 동안은 딥링크 기능이 정상적으로 유지되도록 지원합니다. 다만, 유예 기간이 종료된 후에는 서비스 기능이 제한될 수 있으므로, 대시보드의 알림을 확인하시어 원활한 서비스 이용을 준비해 주시기를 권장합니다.

Firebase Dynamic Links에서 마이그레이션하려면?

기존 FDL 링크 구조를 ViaLink 다이나믹 파라미터로 매핑하는 마이그레이션 가이드를 제공합니다.

기존에 사용하던 외부 서비스의 링크를 그대로 유지하며 옮길 수 있나요?

네, 가능합니다. Vialink의 '커스텀 도메인' 기능을 활성화하면, 기존 타사 서비스에서 사용하던 링크 URL을 변경 없이 그대로 유지할 수 있습니다. 또한, 원활한 마이그레이션을 위해 링크 대량 등록 API와 CSV 임포트 기능을 지원하고 있어, 많은 수의 링크도 간편하게 이전하실 수 있습니다.

개인정보/핑거프린팅은 규정을 준수하나요?

IDFA/GAID를 수집하지 않는 확률적 매칭 방식으로, GDPR·개인정보보호법 스탠스를 보안 백서에 공개하고 있습니다.
