4 things engineers want to verify

Does it work inside in-app browsers?

iOS Universal Links and Android App Links are supported out of the box. In-app browsers such as KakaoTalk, Instagram, and LINE are detected automatically and routed down the optimal path for each environment (including scheme fallback). ViaLink hosts apple-app-site-association and assetlinks.json for you.

How do you match users who don't have the app?

Deferred deep linking: the device signature at click time is stored server-side, and on first launch after install, fingerprint matching restores the original destination screen. The matching window is 72 hours. It works without IDFA/GAID, so it remains valid under ATT.

How do I connect clicks all the way to purchases?

Drop in payment attempt and success events with a single track() call, and click → install → payment attempt → payment success is attributed as one pipeline on a last-touch basis. Compare conversion funnels by channel, campaign, and link right on the dashboard.

ViaLinkSDK.shared.onDeepLink = { link in router.navigate(to: link.path, params: link.params) } ViaLinkSDK.shared.track("purchase_success", data: ["amount": 12900])
What about dynamic URLs like referral links?

Dynamic parameters in the form of /invite/:token are supported. No need to issue a new link every time — the server extracts and records the :token value automatically and passes it to the SDK callback. Ideal for referral and personalized links.

// /invite/:token → extracted automatically vialink.app/invite/FRIEND2026 // received in the SDK callback link.params["token"] // "FRIEND2026"

Last questions before you start

Do deep links stop working the moment my credits run out?

No, not immediately. Even after your credits are exhausted, we provide a generous grace period so your service keeps running smoothly. During that period we notify you that a top-up is needed, and deep link functionality continues to work for a set time. Once the grace period ends, features may be limited — so please keep an eye on your dashboard notifications.

How do I migrate from Firebase Dynamic Links?

We provide a migration guide that maps your existing FDL link structure onto ViaLink dynamic parameters.

Can I keep the links I used with another service?

Yes. With ViaLink's custom domain feature, you can keep your existing third-party link URLs unchanged. We also support a bulk link registration API and CSV import, so migrating even a large number of links is easy.

Is the privacy/fingerprinting approach compliant?

We use probabilistic matching that collects no IDFA/GAID, and our stance on GDPR and privacy law is published in the security whitepaper.

What happens after my free credits run out?

The same rule applies: 1 credit per completed app open. It's a prepaid top-up model, so there is no such thing as a surprise bill.
